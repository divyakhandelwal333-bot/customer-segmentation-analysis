# customer-segmentation-analysis
Customer segmentation analysis using Python, K-Means clustering and Power BI

This project analyzes customer behavior using Python and K-Means clustering.

## Tools Used
Python
Pandas
Scikit-learn
Power BI 

## 🎯 Business Recommendations
Based on customer segmentation analysis:

### 🟢 High-Value Customers
- Premium memberships and VIP experiences
- Early access to high-value products
- Dedicated customer support

### 🔵 Loyal Customers
- Loyalty rewards and referral programs
- Personalized recommendations
- Community engagement initiatives

### 🟡 Regular Customers
- Upsell and cross-sell campaigns
- Bundle offers and subscriptions
- Purchase reminders

### 🔴 At-Risk Customers
- Re-engagement campaigns
- Discount incentives
- Feedback collection

### 🔥 Key Opportunity Segment:
High-income, low-spending customers identified as high-potential group for targeted conversion strategies.
